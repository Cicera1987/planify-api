# Planify API

## Descrição

Este projeto é uma API para gerenciamento de agendamentos, permitindo que usuários façam reservas e administrem seus serviços de forma eficiente. A API é construída utilizando Node.js e Express, e se conecta a um banco de dados MongoDB para armazenar as informações dos usuários e seus agendamentos.

## Pré-requisitos

Certifique-se de ter o seguinte instalado:

- [Node.js](https://nodejs.org/) (versão 14 ou superior)
- [Docker](https://www.docker.com/) (opcional, se você quiser rodar com contêineres)

## Instalação

1. Clone este repositório:

   ```bash
   git clone https://github.com/seu_usuario/planify-api.git
   cd planify-api

2. Instale as dependência:
   npm install

3. Crie o arquivo .env
   PORT=3000
   MONGODB_URI=mongodb://localhost:27017/planify

   # Excutando o Projeto
   npm run dev
